# In-progress agent systems

This directory stores specs and test results for agent systems currently being designed or built.

Each subdirectory corresponds to one agent system under construction:

```
in-progress/
  <prompt-system-name>/
    spec.md           ← design spec (created by /promptsystem:design)
    test-results.md   ← validation results (created by /promptsystem:test)
```

Files here are working state — safe to delete once you've submitted your agent system to the registry.
