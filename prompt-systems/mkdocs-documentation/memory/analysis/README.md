# Analysis artifacts

This directory is populated by `/mkdocs-documentation:analyze` and consumed by `/mkdocs-documentation:build`.

Do not edit these files manually unless you want to adjust the documentation plan before building.

| File | Created by | Purpose |
|------|-----------|---------|
| `glossary.md` | analyze | Domain terms with definitions |
| `acronyms.md` | analyze | Acronyms and their expansions |
| `hierarchy.md` | analyze | Full MkDocs nav outline and page source mapping |
| `tasks.md` | analyze | Prioritized, ordered documentation task checklist |

These files are working state — safe to delete if you want to re-run the analysis from scratch.
